​🚀 IT Lifecycle & SDLC Management Dashboard
​Este projeto é um painel interativo para gestão técnica de projetos de TI, desenvolvido para aplicar os conceitos do curso de Gestão de Projetos de TI (120h).
​🎯 Funcionalidades Principais
​Kanban Dinâmico: Fluxo completo de SDLC (Backlog, Dev, Deploy).
​Métricas Visuais: Gráficos de desempenho e status da Sprint em tempo real.
​Priorização Ágil: Sistema de tags para urgência e tipos de requisitos (Feature, Bug, Infra).
​Persistência Local: Dados salvos via LocalStorage.
​💻 Tecnologias Utilizadas
​HTML5 & CSS3 (Estrutura e Estilo)
​JavaScript ES6+ (Lógica de Drag and Drop e Gráficos)
​Bootstrap 5 & Tailwind CSS (UI/UX Responsiva)
​Chart.js (Visualização de Dados)
